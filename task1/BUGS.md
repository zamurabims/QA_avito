# Список багов по приоритетам

## P0 - критические

**Баг 1**: Некорректная работа фильтра по городу
**Описание:** При выбранном фильтре "Москва" в результатах отображается объявление из города Новосибирск  
**Приоритет:** P0  
**Обоснование:** Пользователь ожидает релевантные результаты, и нарушение этого поведения делает поиск бесполезным  



**Баг 2**: Некорректная работа фильтра "Цена"
**Описание:** При установленном диапазоне цен (1000–30000 ₽) в выдаче отображаются объявления с ценами вне диапазона (например, 500 ₽ и 40000 ₽)  
**Приоритет:** P0  
**Обоснование:** Ценовой фильтр напрямую влияет на выбор пользователя, поэтому некорректная работа ломает основной сценарий поиска  


**Баг 15**: Некорректная работа фильтра по городам
**Описание:** Отображается фильтр "Сначала из Можайска", но город выбран "Москва"  
**Приоритет:** P0  
**Обоснование:** Противоречивая информация в фильтрах подрывает доверие и мешает корректному поиску  



##  P1 - высокий приоритет

**Баг 3**: Некорректная работа фильтра "Пол"  
**Описание:** При выборе фильтра "Мальчик" в выдаче присутствуют объявления с указанием "девочка"  
**Приоритет:** P1  
**Обоснование:** Фильтр не выполняет свою функцию, пользователь получает нерелевантные результаты  



**Баг 4**:Некорректная работа фильтра "Порода"
**Описание:** Выбрана порода "Персидская", однако в результатах отображаются другие породы (например, экзотическая короткошерстная)  
**Приоритет:** P1  
**Обоснование:** Один из ключевых критериев выбора работает некорректно, что приводит к потере релевантности результатов  



**Баг 5**: Некорректное содержимое фильтра "Возраст"
**Описание:** В категории "Кошки" в фильтре отображается значение "Щенок"  
**Приоритет:** P1  
**Обоснование:** Логическая ошибка и смешение категорий вводят пользователя в заблуждение и снижают доверие к сервису  



**Баг 6**: Несоответствие категории объявления
**Описание:** В выдаче отображается объявление с мягкой игрушкой вместо живого котенка 
**Приоритет:** P1  
**Обоснование:** Ошибка категоризации приводит к нерелевантной выдаче и нарушает пользовательский сценарий  


## P2 - средний приоритет

**Баг 7**: Несоответствие количества объявлений
**Описание:** Указано "Показать 8 объявлений", однако фактически отображается 19  
**Приоритет:** P2  
**Обоснование:** Пользователь получает противоречивую информацию о результатах, что снижает доверие к системе  



**Баг 8**: Некорректная работа блока "Подобрали для вас"
**Описание:**  
- отображается коллаж вместо стандартной карточки; 
- количество изображений превышает допустимое;
- отсутствует стрелка для прокрутки.

**Приоритет:** P2  
**Обоснование:** Блок рекомендаций работает некорректно и ухудшает пользовательский опыт  



**Баг 9**: Отображение системного сообщения об ошибке
**Описание:** На странице отображается сообщение "Ошибка, обновите страницу" без явной причины  
**Приоритет:** P2  
**Обоснование:** Пользователь может воспринять страницу как неработающую и покинуть сайт  


**Баг 10**:Сдвиг верстки карточек
**Описание:** Во втором ряду карточек наблюдается смещение изображений  
**Приоритет:** P2  
**Обоснование:** Визуальный дефект ухудшает восприятие и удобство просмотра  



**Баг 13**: Отсутствие параметра "Окрас"
**Описание:** В списке фильтров отсутствует параметр "Окрас"  
**Приоритет:** P2  
**Обоснование:** Ограничивает возможности фильтрации и ухудшает пользовательский опыт  



**Баг 14**: Отсутствие кнопки "Все категории"
**Описание:** В навигации отсутствует ожидаемая кнопка возврата "Все категории"  
**Приоритет:** P2  
**Обоснование:** Усложняет навигацию, особенно для новых пользователей  



**Баг 16**: Некорректная подсказка у кнопки "Избранное"
**Описание:** При наведении отображается текст "Не показывайте мне похожие объявления"  
**Приоритет:** P2  
**Обоснование:** Несоответствие текста вводит пользователя в заблуждение  



##  P3 - низкий приоритет

**Баг 11**: Грамматическая ошибка "Коръера в Авито"
**Описание:** В тексте присутствует ошибка в слове "Карьера"  
**Приоритет:** P3  
**Обоснование:** Снижает доверие к сервису и влияет на восприятие качества  



**Баг 12**: Грамматическая ошибка "Уведомять о новых"
**Описание:** В кнопке присутствует опечатка  
**Приоритет:** P3  
**Обоснование:** Ухудшает качество интерфейса  



**Баг 17**: Некорректная работа курсора при наведении на избранное
**Описание:** При удержании курсора на иконку "избранное" не меняется его стиль (остается статичным)  
**Приоритет:** P3  
**Обоснование:** Визуальный дефект, не влияющий на функциональность  



**Баг 18**: Некорректный порядок в навигационной цепочке
**Описание:** В навигационной цепочке нарушена логика вложенности: «Главная > Животные > Персидская > Кошки». Подкатегория идет перед основной категорией  
**Приоритет:** P3  
**Обоснование:** Навигационная ошибка, но не блокирует использование  

**Баг 19**: Не отображается состояние выбранного значения в фильтре “Порода”
**Описание:** После выбора значения “Персидская” в фильтре “Порода” чекбокс не отображается как выбранный
**Приоритет:** P3  
**Обоснование:** Навигационная ошибка, но не блокирует использование
